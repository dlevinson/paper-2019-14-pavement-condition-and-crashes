# Legacy Pavement/QGIS Code

This directory contains two legacy Python/QGIS-era scripts found in the local source folder for `paper-2019-14`. They appear to support the paper's pavement/crash joining and model-ready summary construction.

The scripts contain hard-coded original workstation paths and are not standalone runnable without path adaptation and separately obtained MnDOT/MetroGIS inputs. No source data are included.
